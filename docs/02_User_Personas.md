# 👤 User Personas

## Persona 1: huda shaikh  (Primary Persona)

### Profile
- **Age:** 21
- **Occupation:** Final-year BSc IT Student
- **Location:** Mumbai, India

### About
Aisha is a university student balancing academics, certifications, internship applications, personal projects, and extracurricular activities. She captures ideas in multiple places, making it difficult to stay organized and revisit valuable information.

### Goals
- Keep all ideas in one place
- Organize academic notes efficiently
- Save project inspirations
- Track personal learning
- Quickly find past information

### Pain Points
- Notes are scattered across different apps
- Frequently forgets where information was saved
- Loses valuable ideas over time
- Difficulty connecting related thoughts

### How Echo Helps
Echo provides a centralized workspace where huda can capture ideas, connect related thoughts, and build a personal knowledge system.

# 🎯 Common User Needs

Across all personas, users need:

- A single place to capture ideas.
- Fast search and retrieval.
- Visual connections between related thoughts.
- Simple and distraction-free interface.
- Secure cloud-based storage.
